## Description

The CLI's `init` command requires installing `@mabulu-inc/simplicity-admin-cli` before scaffolding a project. The standard Node.js pattern for project scaffolding is a `create-*` package that works with `npm create` / `pnpm create` / `npx` without prior installation:

```bash
pnpm create @mabulu-inc/simplicity-admin my-app
npx @mabulu-inc/create-simplicity-admin my-app
```

Create a new `packages/create` workspace package that wraps the existing `init` logic from `@mabulu-inc/simplicity-admin-cli`. The CLI `init` command should remain as-is for existing users.

Update all documentation (README, docs-site, specs) to use the `create` command as the primary getting-started path.
