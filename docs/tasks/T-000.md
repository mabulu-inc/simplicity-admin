# T-000: Infrastructure bootstrap

- **Status**: DONE
- **Milestone**: 0 — Infrastructure
- **Depends**: none
- **PRD Reference**: §1

## Description

Set up the TypeScript project infrastructure with quality tooling.

**Setup:**

- Initialize TypeScript project with pnpm
- Configure Vitest for testing
- Set up ESLint for linting
- Set up Prettier for formatting
- Configure husky + lint-staged for pre-commit hooks
- Create `pnpm check` script that runs: lint → format → typecheck → build → test:coverage

**Database:**

- Set up Docker Compose for PostgreSQL via Docker
- Configure connection and health check

## Produces

- Project configuration files
- Vitest configuration
- Quality check script (`pnpm check`)
- ESLint + Prettier configuration
- husky + lint-staged setup
- Docker Compose configuration
- A passing `pnpm check` run
